# 写作套装 - 30天行动指南

## 🎯 你的目标

完成本计划后，你将能够：
- ✅ **第3天**：使用模板写出第一篇完整文章
- ✅ **第7天**：建立稳定的写作习惯（每天500字）
- ✅ **第14天**：完成第5篇文章，找到自己的写作节奏
- ✅ **第30天**：累计完成10篇文章，建立个人写作系统

---

## 📋 第一周：快速上手（Day 1-7）

### Day 1：认识你的工具（时间：60分钟）

**📚 任务清单：**
- [ ] 阅读《写作模板使用指南》（20分钟）
- [ ] 浏览所有30+写作模板，了解每个模板的用途（15分钟）
- [ ] 选择3个最适合你的模板，保存到常用目录（10分钟）
- [ ] 完成"模板测试练习"（15分钟）

**💡 今日重点**：不需要完美，先熟悉工具

**📝 今日输出**：在模板测试区写100字介绍自己

---

### Day 2：第一篇文章（时间：90分钟）

**📚 任务清单：**
- [ ] 选择一个"新手友好型"模板（推荐：观点文模板）（10分钟）
- [ ] 选择一个你熟悉或感兴趣的主题（10分钟）
- [ ] 按照模板大纲填充内容（50分钟）
- [ ] 自己通读一遍，做简单修改（20分钟）

**💡 今日重点**：先完成，再完美

**📝 今日输出**：你的第一篇文章初稿（800-1000字）

---

### Day 3：修改优化（时间：60分钟）

**📚 任务清单：**
- [ ] 使用《爆款标题库》优化标题（15分钟）
- [ ] 检查文章是否符合"模板检查清单"（15分钟）
- [ ] 朗读文章，修改拗口的句子（20分钟）
- [ ] 加上合适的配图建议（10分钟）

**💡 今日重点**：好文章是改出来的

**📝 今日输出**：第一篇文章定稿

---

### Day 4：发布文章（时间：40分钟）

**📚 任务清单：**
- [ ] 选择发布平台（公众号/知乎/头条/简书）（5分钟）
- [ ] 按照平台排版要求调整格式（20分钟）
- [ ] 添加封面图和作者简介（10分钟）
- [ ] 正式发布！记录发布时间（5分钟）

**💡 今日重点**：敢于发布是第一步

**📝 今日输出**：完成第一篇文章发布

---

### Day 5：复盘学习（时间：30分钟）

**📚 任务清单：**
- [ ] 记录发布后的数据（阅读、点赞、评论）（5分钟）
- [ ] 分析：哪部分最成功？哪部分可以改进？（15分钟）
- [ ] 写在"写作日志"中（10分钟）

**💡 今日重点**：持续改进的关键

**📝 今日输出**：第一篇文章复盘笔记

---

### Day 6：休息消化（时间：30分钟）

**📚 任务清单：**
- [ ] 阅读一篇优秀的同类文章（15分钟）
- [ ] 分析它用了哪些技巧（10分钟）
- [ ] 记录3个可以借鉴的点（5分钟）

**💡 今日重点**：输入决定输出质量

**📝 今日输出**：3个写作技巧笔记

---

### Day 7：每周总结（时间：45分钟）

**📚 任务清单：**
- [ ] 本周写作字数统计（5分钟）
- [ ] 本周成果：发布了X篇文章（5分钟）
- [ ] 本周最大的收获是什么？（10分钟）
- [ ] 本周遇到的困难是什么？（10分钟）
- [ ] 下周目标是什么？（15分钟）

**💡 今日重点**：定期总结才能持续进步

**📝 今日输出**：第一周总结报告

---

## 📋 第二周：建立习惯（Day 8-14）

### Day 8-10：连续写作挑战

**📚 每日任务（重复3天）：**
- [ ] 选择一个模板（5分钟）
- [ ] 设定50分钟专注写作时间
- [ ] 目标：500字，不追求完美
- [ ] 记录写作日志（10分钟）

**💡 挑战规则**：
- 不能跳过，即使只能写100字
- 写完才能玩手机
- 每天选择不同模板练习

---

### Day 11：模板进阶（时间：60分钟）

**📚 任务清单：**
- [ ] 尝试一个新的模板类型（如对比文模板）（10分钟）
- [ ] 使用"对比检查清单"验证结构（10分钟）
- [ ] 完成文章（40分钟）

**💡 今日重点**：掌握多种写作方式

---

### Day 12：标题专项训练（时间：45分钟）

**📚 任务清单：**
- [ ] 翻看《爆款标题库》（10分钟）
- [ ] 为最近写的3篇文章各取5个不同标题（20分钟）
- [ ] 请朋友/社群成员投票选出最好的（15分钟）

**💡 今日重点**：好标题让文章传播力提升10倍

---

### Day 13：配图和排版（时间：40分钟）

**📚 任务清单：**
- [ ] 学习《配图指南》（10分钟）
- [ ] 为本周文章添加配图（20分钟）
- [ ] 优化排版（10分钟）

**💡 今日重点**：视觉效果决定点击率

---

### Day 14：半月总结（时间：60分钟）

**📚 任务清单：**
- [ ] 本周完成：X篇文章，X字
- [ ] 对比Day 7的总结，你的进步是什么？
- [ ] 当前最擅长的模板是？
- [ ] 下半个月的写作计划是什么？

**💡 今日重点**：庆祝进步，继续前行

---

## 📋 第三周：突破瓶颈（Day 15-21）

### Day 15-16：深度写作

**📚 任务**：
- 用2天时间完成一篇深度长文（2000-3000字）
- 使用"长文模板"
- 要求：有独特观点，有数据支撑

---

### Day 17：阅读充电日

**📚 任务**：
- 阅读2篇优质长文
- 分析其结构、论证方式、语言风格
- 记录在"写作灵感库"

---

### Day 18：用户反馈收集

**📚 任务**：
- 给最近文章的评论者回复
- 询问读者喜欢什么内容
- 记录3个改进方向

---

### Day 19-21：系列文章挑战

**📚 任务**：
- 选择一个主题（如"时间管理"）
- 用3天写完系列文章（上中下）
- 使用"系列文模板"

---

## 📋 第四周：形成系统（Day 22-30）

### Day 22-24：速度训练

**📚 任务**：
- 挑战：30分钟内完成500字文章
- 连续3天，每天2篇
- 使用最熟练的模板

### Day 25：复盘所有文章

**📚 任务**：
- 回看第1-24天写的所有文章
- 找出你最好的3篇
- 分析为什么这3篇最好？

---

### Day 26-28：个人化模板

**📚 任务**：
- 选择最常用的模板
- 根据自己习惯修改优化
- 创建"我的专属模板"

---

### Day 29：制定长期计划

**📚 任务**：
- 回顾30天成果
- 制定未来90天写作计划
- 设定：每月目标、总字数目标、平台目标

---

### Day 30：庆祝与分享

**📚 任务**：
- 统计30天成果：文章数、总字数、最高阅读
- 在社群分享你的成长故事
- 鼓励其他正在坚持的人

- [ ] 标题是否吸引人（参考标题库）

- [ ] 结尾是否有行动号召
- [ ] 配图是否合适
- [ ] 排版是否美观

### 每日习惯检查：
- [ ] 今天写作了吗？
- [ ] 记录写作日志了吗？
- [ ] 记录灵感了吗？

## 🆘 遇到问题怎么办？

### Q1: 没有灵感怎么办？
- 翻看《灵感收集清单》
- 使用"观点文模板"从热点话题切入
- 在社群里提问

### Q2: 写不出来怎么办？
- 降低目标：从500字降到200字
- 使用"流水账模板"先开始
- 提醒自己：先完成再完美

### Q3: 没人看怎么办？
- 坚持30天，数据会慢慢积累
- 在社群互动，提升曝光
- 优化标题和封面

### Q4: 坚持不下去怎么办？
- 加入打卡群，互相督促
- 每周给自己小奖励
- 回顾Day 1的目标
