5. 替换以下文件中的 `G-XXXXXXXXXX`：

### 2️⃣ Mailchimp 邮件订阅配置

#### 步骤：
1. 访问 https://mailchimp.com/
2. 点击"Sign Up Free"注册
3. 完成账号设置和邮箱验证
4. 创建受众（Audience）：
   - 受众名称：Newsletter Subscribers
   - 添加字段：Email, Name (FNAME)
5. 创建注册表单：
   - Audience → Manage Audience → Signup Forms
   - 选择 "Embedded Forms"
   - 选择 "Naked" 或 "Super Slim" 样式
   - 复制 form action URL（格式：https://<your-account>.list-manage.com/subscribe/post?u=...）
6. 替换 `subscribe.html` 中的 URL：

#### 需要替换的文件：
- [ ] `subscribe.html` (第 266 行)
- [ ] 将 `YOUR_MAILCHIMP_FORM_URL_HERE` 替换为真实的 form action URL

#### 验证方法：
```
1. 访问 subscribe.html
2. 填写邮箱并提交
3. 检查 Mailchimp 受众列表
4. 应该看到新的订阅者
```

### 4️⃣ 部署 SEO 文件

#### 步骤：
1. 将以下文件上传到网站根目录：
   - [ ] `robots.txt`
   - [ ] `sitemap.xml`

2. 提交 sitemap 到 Google Search Console：
   - 访问 https://search.google.com/search-console/
   - 添加你的网站
   - 验证所有权
   - 进入"网站地图" (Sitemaps)
   - 输入：`https://thecolin.vip/sitemap.xml`
   - 点击"提交"

#### 验证方法：
```
1. 访问 https://thecolin.vip/robots.txt
2. 应该看到 robots.txt 内容
3. 访问 https://thecolin.vip/sitemap.xml
4. 应该看到 sitemap.xml 内容
5. 在 Google Search Console 查看 sitemap 状态
```

---

### 5️⃣ 域名和 HTTPS 配置（如果需要）

#### 步骤：
1. 购买域名（如果还没有）
2. 配置 DNS 记录指向你的服务器
3. 配置 HTTPS 证书（Let's Encrypt 免费）
4. 确保所有链接使用 `https://`

#### 验证方法：
```
1. 访问 https://thecolin.vip
2. 浏览器地址栏应显示锁图标
3. 使用 https://www.ssllabs.com/ssltest/ 测试 SSL 评分
```
