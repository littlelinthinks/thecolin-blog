## 📝 如何添加新文章

### 方法1: 使用Markdown文件(推荐)

1. **创建Markdown文件**
   ```
   # 文章标题

   文章内容...

   ---
   *发布时间：2026年1月18日*
   ```

2. **保存到`/articles`文件夹**
   - 文件名格式: `W{周数}_{日期}_星期{几}_栏目_标题.md`
   - 例如: `W4_0118_周四_小Lin读_觉醒方法论.md`

3. **重新生成`articles.json`**
   ```bash
   python3 /Users/colinying/Desktop/generate_articles_json.py
   ```

4. **刷新网页**即可看到新文章

### 方法2: 直接修改`articles.json`

1. 打开`articles.json`
2. 添加新文章对象:
   ```json
   {
     "id": "article-unique-id",
     "title": "文章标题",
     "titleEn": "Article Title",
     "filename": "文章文件名.md",
     "category": "thinking",
     "categoryEn": "Deep Thinking",
     "categoryName": "小Lin析",
     "date": "2026年1月18日",
     "dateEn": "January 18, 2026",
     "excerpt": "文章摘要...",
     "excerptEn": "Article excerpt...",
     "readTime": "5分钟阅读",
     "readTimeEn": "5 min read"
   }
   ```
3. 保存并刷新网页

### 工作流程

1. **页面加载**
   ```
   页面加载 → 读取articles.json → 生成文章卡片 → 显示前6篇
   ```

2. **用户点击"Load More"**
   ```
   显示6篇文章 → 更新计数器 → 检查是否全部显示
   ```

3. **用户点击文章**

4. **用户搜索**
   ```
   输入关键词 → 过滤allArticles数组 → 显示搜索结果 → 点击跳转
   ```

## ⚠️ 注意事项

1. **文件名编码**
   - 确保Markdown文件名使用UTF-8编码
   - 避免特殊字符

2. **文章ID唯一性**
   - 每个文章必须有唯一的`id`
   - 建议: 使用文件名转换为ID

3. **Markdown格式**
   - 使用标准Markdown语法
   - 文章开头应该有标题(`# 标题`)

4. **图片路径**
   - 如果文章包含图片,使用相对路径
   - 例如: `![图片](images/example.jpg)`

## 📞 常见问题

### Q1: 为什么文章没有显示?

A: 检查以下几点:
1. `articles.json`是否正确生成
2. 文章ID是否唯一
3. 浏览器控制台是否有错误

### Q2: 如何修改文章顺序?

A: 在`articles.json`中调整数组顺序,最新的放在最前面。

### Q3: 如何批量更新文章?

A: 使用Python脚本批量处理:
```python
import json

with open('articles.json', 'r', encoding='utf-8') as f:
    articles = json.load(f)

# 批量修改...

with open('articles.json', 'w', encoding='utf-8') as f:
    json.dump(articles, f, ensure_ascii=False, indent=2)
```

### Q4: 如何添加图片到文章?

A:
1. 将图片放在`/articles/images/`文件夹
2. 在Markdown中使用: `![图片描述](images/filename.jpg)`
