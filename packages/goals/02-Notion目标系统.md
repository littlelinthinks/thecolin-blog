## 一、Notion系统架构

### 整体结构
```
📁 个人成长系统/
├── 📊 仪表盘（Dashboard）
├── 🎯 目标数据库
├── ✅ 任务数据库
├── 🔄 习惯数据库
├── 📅 日志数据库
└── 📚 资源库
```

### 数据库关系
```
目标数据库
    ↓ 关联
任务数据库

习惯数据库
    ↓ 独立
每日打卡

日志数据库
    ↓ 关联
任务/习惯
```

- ✅ 已完成（绿色）
- ⏸️ 暂停（黄色）

**里程碑示例：**

## 四、使用教程

### 步骤1：创建数据库

**在Notion中：**
1. 新建页面，命名为"个人成长系统"
2. 在页面内创建4个数据库：
   - 目标数据库
   - 任务数据库
   - 习惯数据库
   - 日志数据库

**添加属性：**
- 按上面的完整代码添加所有字段
- 选择对应类型和选项

---

### 步骤2：设置关联

**任务 → 目标：**
1. 打开任务数据库
2. 添加"关联目标"属性
3. 选择关联到"目标数据库"

**日志 → 任务：**
1. 打开日志数据库
2. 添加"完成任务"属性
3. 选择关联到"任务数据库"

---

### 步骤3：创建仪表盘

**布局：**
```
# 🎯 个人成长仪表盘

## 📊 本周目标
[嵌入目标数据库视图：过滤本周目标]

## ✅ 今日任务
[嵌入任务数据库视图：截止日期=今天]

## 🔄 今日习惯
[嵌入习惯数据库视图：每日习惯]

## 📝 今日日志
[创建日志按钮]
```

**创建视图：**
- 目标：按截止日期排序，过滤本周
- 任务：按优先级排序，过滤今天
- 习惯：按名称排序，过滤每日

---

### 步骤4：建立工作流

**每日流程：**

**早晨（5分钟）：**
1. 打开仪表盘
2. 查看"今日任务"
3. 查看今日习惯
4. 设定3个重点任务

**白天（工作）：**
1. 完成任务，更新状态
2. 完成习惯，点击打卡

**晚上（10分钟）：**
1. 填写日志
2. 记录能量/情绪
3. 写下明日计划

## 五、自动化设置

### 使用Notion公式

#### 公式1：自动计算进度

```javascript
prop("已完成任务") / prop("总任务数") * 100
```

#### 公式2：优先级标记

```javascript
if(prop("优先级") == "🔴 重要且紧急", "🔴",
  if(prop("优先级") == "🔵 重要不紧急", "🔵",
    if(prop("优先级") == "🟡 紧急不重要", "🟡", "⚪")))
```

#### 公式3：连续天数计算

```javascript
if(prop("上次打卡") == now(), prop("连续天数") + 1, 0)
```

- ⚪ 普通：白色

- 🏗️ 执行：需要执行

- 📖 学习：需要学习

### 技巧4：公式提醒

**过期任务提醒：**
```javascript
if(prop("截止日期") < now() and prop("状态") != "已完成", "⚠️ 已过期", "")
```

**优先任务：**
```javascript
if(prop("优先级") == "🔴 重要且紧急", "🔴 优先", "")
```
