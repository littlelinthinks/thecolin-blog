## 🎯 决策矩阵工具箱

### 工具1：加权决策矩阵

**用途：** 多选项、多维度比较

**步骤：**

#### Step 1: 列出决策选项
```
选项A：继续当前工作
选项B：跳槽到新公司
选项C：自己创业
```

#### Step 2: 确定评价维度（3-5个）
```
维度1：收入潜力
维度2：工作稳定性
维度3：个人成长
维度4：工作生活平衡
维度5：兴趣爱好
```

#### Step 3: 为每个维度设置权重（总分100）
```
收入潜力：30%
工作稳定性：20%
个人成长：25%
工作生活平衡：15%
兴趣爱好：10%
```

#### Step 4: 为每个选项评分（1-10分）

| 维度 | 权重 | 选项A | 选项B | 选项C |
|------|------|-------|-------|-------|
| 收入潜力 | 30% | 6分 | 8分 | 9分 |
| 工作稳定性 | 20% | 9分 | 7分 | 3分 |
| 个人成长 | 25% | 5分 | 7分 | 9分 |
| 工作生活平衡 | 15% | 8分 | 6分 | 4分 |
| 兴趣爱好 | 10% | 4分 | 6分 | 10分 |

#### Step 5: 计算加权得分

```
选项A得分：
= 6×30% + 9×20% + 5×25% + 8×15% + 4×10%
= 1.8 + 1.8 + 1.25 + 1.2 + 0.4
= 6.45分

选项B得分：
= 8×30% + 7×20% + 7×25% + 6×15% + 6×10%
= 2.4 + 1.4 + 1.75 + 0.9 + 0.6
= 7.05分

选项C得分：
= 9×30% + 3×20% + 9×25% + 4×15% + 10×10%
= 2.7 + 0.6 + 2.25 + 0.6 + 1
= 7.15分
```

**结论：** 选项C得分最高（7.15），建议选择创业

### 工具2：SWOT分析法

**用途：** 分析选择的优势和劣势

**格式：**

```
        内部因素         外部因素
积极面 | 优势（S）       机会（O）
       | • 专业能力强    | • 行业发展快
       | • 资源丰富      | • 政策支持
       |                |
消极面 | 劣势（W）       威胁（T）
       | • 资金不足      | • 竞争激烈
       | • 经验缺乏      | • 市场变化快
```

**分析策略：**

| 策略 | 方法 |
|------|------|
| SO策略 | 利用优势抓住机会 |
| ST策略 | 利用优势应对威胁 |
| WO策略 | 克服劣势抓住机会 |
| WT策略 | 减少劣势和威胁 |

**示例：创业决策**

```
优势（S）：
• 有行业经验
• 有技术能力
• 有团队资源

劣势（W）：
• 启动资金不足
• 管理经验缺乏
• 品牌知名度低

机会（O）：
• 市场需求增长
• 政策支持创新
• 技术门槛降低

威胁（T）：
• 大公司进入
• 价格战激烈
• 技术更新快

策略（SO）：利用技术能力抓住市场机会
策略（ST）：建立技术壁垒应对竞争
策略（WO）：寻找合作伙伴补充资金和管理
策略（WT）：先做细分市场降低风险
```

### 工具3：10-10-10法则

**用途：** 评估决策的长期影响

**问题：**
```
1. 10分钟后，我会怎么想？
2. 10个月后，我会怎么想？
3. 10年后，我会怎么想？
```

**应用：冲动购买决策**

| 时间点 | 思考问题 |
|--------|----------|
| 10分钟后 | 买完会有满足感吗？会后悔吗？ |
| 10个月后 | 还在使用吗？钱花得值吗？ |
| 10年后 | 这个决定对我的人生有什么影响？ |

**结论：** 如果长期看影响不大，说明不是重要决策

### 工具4：机会成本分析

**用途：** 评估不做这件事的损失

**公式：**
```
机会成本 = 放弃的最佳替代方案的价值
```

**示例：时间分配**

```
选择A：刷短视频2小时
机会成本：放弃了2小时可以做的事
├─ 阅读1本书（价值¥100）
├─ 锻炼身体（价值健康）
└─ 学习新技能（价值¥500/天）

机会成本 = ¥100 + 健康 + ¥500 = 高

结论：刷短视频的机会成本很高
```

### 工具5：后悔最小化框架

**用途：** 长期视角避免遗憾

**问题：**
```
当你80岁回顾人生，你会：
1. 因为做了这件事而后悔？
2. 因为没做这件事而后悔？
```

**原则：**
- 做：如果不做会后悔 → 做吧
- 不做：如果做了会后悔 → 别做

**示例：**

| 决策 | 80岁后悔分析 |
|------|-------------|
| 辞职创业 | 如果不做 → 一辈子后悔"如果当初..." |
| 买昂贵包 | 如果做了 → 后悔"浪费钱" |
| 去远方旅行 | 如果不做 → 后悔"没去看看世界" |

## 🎯 决策流程模板

### 完整决策流程

#### Step 1: 明确决策目标（5分钟）
- 我想要什么？

- 时间框架？

- 脑暴所有可能性

#### Step 5: 做出决定（5分钟）
- 综合分析

- 1个月后复盘

## ⚠️ 决策陷阱

### 陷阱1：分析瘫痪

**症状：**
- 过度分析
- 永远觉得信息不够
- 无法做决定

**解决方法：**
- 设定决策时间限制
- "足够好"即可
- 行动 > 完美

---

### 陷阱2：沉没成本陷阱

**症状：**
- 因为已经投入所以继续
- "都花了这么多钱/时间"
- 无法及时止损

**解决方法：**
- 只看未来，不看过去
- "如果今天重新开始，还会做吗？"
- 果断止损

---

### 陷阱3：确认偏误

**症状：**
- 只看支持自己观点的信息
- 忽视反对意见
- 过度自信

**解决方法：**
- 主动寻找反对意见
- 列出"为什么可能错"
- 第三方视角审视

---

### 陷阱4：锚定效应

**症状：**
- 被第一个信息锚定
- 无法客观判断
- 决策偏差

**解决方法：**
- 先收集足够信息
- 从多个角度评估
- 不轻易下结论
