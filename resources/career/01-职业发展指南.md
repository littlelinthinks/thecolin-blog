## 一、职业规划

### 职业规划三问

#### 问题1：我想要什么？
```
思考维度：
1. 兴趣所在
   - 做什么时感到快乐？
   - 什么话题让你兴奋？
   - 愿意花时间学习什么？

2. 价值观
   - 最看重什么？
   - 金钱、影响力、自由、稳定？
   - 不能妥协的原则？

3. 长期愿景
   - 5年后想成为谁？
   - 10年后想在哪里？
   - 退休后希望留下什么？
```

#### 问题2：我有什么？
```
资源盘点：
1. 技能
   - 硬技能：专业知识、技术能力
   - 软技能：沟通、领导、协作

2. 经验
   - 工作经验
   - 项目经验
   - 生活经历

3. 优势
   - 天赋优势
   - 个人特质
   - 获得成就
```

#### 问题3：我能做什么？
```
机会识别：
1. 市场需求
   - 哪些行业在增长？
   - 哪些技能稀缺？
   - 哪些问题需要解决？

2. 职业路径
   - 传统路径：专家、管理、创业
   - 新兴路径：自由职业、远程工作
   - 组合路径：主业+副业

3. 行动计划
   - 短期（1年内）：提升能力
   - 中期（3年内）：获得机会
   - 长期（5年以上）：达成目标
```

---

### SWOT分析

#### 内部因素（S/W）
**优势（Strengths）：**
- 你最擅长什么？
- 什么让你脱颖而出？
- 什么资源你可以利用？

**劣势（Weaknesses）：**
- 哪些方面需要改进？
- 什么限制了你的发展？
- 哪些技能是你的短板？

#### 外部因素（O/T）
**机会（Opportunities）：**
- 行业有哪些趋势？
- 有哪些新岗位出现？
- 有哪些学习机会？

**威胁（Threats）：**
- 行业竞争如何？
- 技术是否可能被取代？
- 经济形势如何影响？

#### 分析与应用
```
SO战略（优势+机会）：
发挥优势抓住机会

WO战略（劣势+机会）：
弥补劣势抓住机会

ST战略（优势+威胁）：
发挥优势应对威胁

WT战略（劣势+威胁）：
减少劣势避免威胁
```

## 二、技能提升

### 技能分类

#### 硬技能
**专业领域：**
- 行业专业知识
- 工具软件能力
- 技术/工程能力
- 语言能力

**可衡量指标：**
- 证书/资质
- 项目经验
- 成功案例

---

#### 软技能
**沟通能力：**
- 口头沟通
- 书面沟通
- 演讲表达

**协作能力：**
- 团队合作
- 冲突解决
- 领导力

**思维能力：**
- 批判性思维
- 解决问题
- 创新思维

---

### 学习方法

#### 方法1：刻意练习
```
刻意练习4要素：
1. 明确目标
   - 想要提升什么？
   - 如何衡量进步？

2. 专注练习
   - 专注时间（45-90分钟）
   - 无干扰环境
   - 反复练习

3. 即时反馈
   - 找导师指导
   - 录像自我观察
   - 与他人对比

4. 逐步挑战
   - 超出舒适区
   - 稍微困难
   - 循序渐进
```

#### 方法2：70/20/10原则
```
学习内容分配：
70% - 工作中学习
   - 实际项目应用
   - 问题解决实践
   - 同事交流学习

20% - 正式学习
   - 课程/培训
   - 书籍阅读
   - 系统化学习

10% - 社交学习
   - 会议/论坛
   - 社交网络
   - 行业活动
```

#### 方法3：费曼学习法
```
步骤：
1. 选择学习主题
2. 用简单语言解释
3. 找出理解不深的地方
4. 重新学习简化
5. 教学巩固
```

## 三、简历优化

### 简历核心原则

#### 原则1：结果导向
```
错误写法：
"负责销售工作"

正确写法：
"负责销售工作，年销售额500万，同比增长30%"
```

#### 原则2：量化成就
```
使用数字：
- 效率提升：节省40%时间
- 成本降低：减少50%预算
- 规模影响：服务10,000+客户
```

#### 原则3：关键词匹配
```
分析JD关键词：
- 提取职位描述关键词
- 调整简历用词
- 提高ATS系统匹配度
```

---

### 简历结构

#### 模板
```
[姓名]
[联系方式]

职业摘要
[2-3句话总结，突出核心价值]

工作经验
[公司名称] | [职位] | [时间]
- [成就1，量化]
- [成就2，量化]
- [成就3，量化]

[其他公司经历，格式相同]

教育背景
[学校名称] | [专业] | [时间]
- [荣誉/奖项]
- [相关课程]

技能
[分类技能]
- [技能1] - [熟练度]
- [技能2] - [熟练度]

项目
[项目名称] | [时间]
- [项目描述]
- [你的贡献]
- [项目成果]
```

---

### 简历优化技巧

#### 技巧1：动词开头的成就描述
```
强力动词：
领导：带领、主导、组织
创造：创建、开发、设计
改进：优化、提升、降低
解决：解决、克服、实现

示例：
"带领5人团队，完成3个项目，提升效率30%"
"开发新流程，降低错误率40%"
```

#### 技巧2：STAR法则
```
Situation（情境）：
"公司需要提升客户满意度"

Task（任务）：
"负责改进客户服务流程"

Action（行动）：
"设计并实施新的反馈系统，培训团队"

Result（结果）：
"客户满意度从70%提升到90%"
```

#### 技巧3：定制简历
```
针对每个职位：
1. 分析职位描述
2. 提取关键要求
3. 调整简历内容
4. 突出匹配经验
```

## 四、面试技巧

### 面试准备

#### 准备1：公司研究
```
研究维度：
1. 公司业务
   - 核心产品/服务
   - 市场定位
   - 竞争对手

2. 公司文化
   - 价值观/使命
   - 团队风格
   - 工作环境

3. 行业趋势
   - 行业挑战
   - 未来机会
   - 热点话题
```

#### 准备2：常见问题准备
**自我介绍：**
```
1分钟版本：
"我是[姓名]，[年经验]。最擅长[技能1]和[技能2]。之前在[公司]负责[工作]，取得了[成就]。现在希望加入贵公司，[目标]。"
```

**优势/劣势：**
```
优势：
- 选择2-3个与职位匹配的优势
- 用例子证明

劣势：
- 选择真实但不致命的劣势
- 说明如何改进

示例劣势：
"我的公开演讲需要提升。现在我会积极参加培训，并主动申请演讲机会。"
```

#### 准备3：准备提问
```
向面试官提问：
1. 团队结构如何？
2. 这个岗位的成功标准是什么？
3. 团队目前面临的挑战是什么？
4. 公司的未来计划是什么？
5. 为什么这个岗位现在开放？
```

---

### 面试技巧

#### 技巧1：STAR法则回答
```
示例问题："请描述一个你解决过的难题"

S-情境：
"在项目中，客户临时要求在2天内完成紧急任务"

T-任务：
"我的任务是在不牺牲质量的前提下，按时交付"

A-行动：
"我立即重新排优先级，协调资源，加班工作，并与客户保持实时沟通"

R-结果：
"按时完成任务，客户满意，公司获得了后续订单"
```

#### 技巧2：非语言沟通
```
眼神接触：
- 自然、适度
- 表达兴趣和自信

姿态：
- 身体前倾
- 表现出热情

手势：
- 适度使用
- 强调关键点

声音：
- 清晰、有节奏
- 自信的语调
```

#### 技巧3：应对困难问题
```
不知如何回答：
1. 礼貌请求思考时间
2. 使用相关经验
3. 说明愿意学习

薪资谈判：
1. 提问公司预算
2. 陈述市场价值
3. 准备具体数字

失败经历：
1. 承认失败
2. 解释原因
3. 学到的教训
4. 改进措施
```

## 五、职场进阶

### 职场晋升策略

#### 策略1：建立影响力
```
影响力来源：
1. 专业能力
   - 成为专家
   - 解决难题
   - 分享知识

2. 人际网络
   - 建立信任
   - 帮助他人
   - 展现领导力

3. 可见性
   - 主动汇报
   - 提出建议
   - 参与决策
```

#### 策略2：管理能力
```
初级管理：
- 任务分配
- 团队协调
- 基础指导

中级管理：
- 战略规划
- 资源管理
- 团队发展

高级管理：
- 文化建设
- 变革管理
- 战略决策
```

---

### 职场沟通

#### 沟通原则
```
向上沟通（对领导）：
- 准备充分
- 结论先行
- 提供解决方案

平级沟通（对同事）：
- 尊重专业
- 坦诚交流
- 互利思维

向下沟通（对下属）：
- 清晰指令
- 反馈及时
- 激励鼓励
```

---

### 职业转换

#### 转换前准备
```
自我评估：
1. 为什么想要转换？
2. 新领域适合我吗？
3. 需要什么新技能？

市场调研：
1. 行业前景如何？
2. 需要什么能力？
3. 如何进入？
```

#### 转换策略
```
路径1：内部转岗
- 优势：熟悉公司
- 方法：与部门领导沟通
- 时间：边工作边准备

路径2：培训转行
- 优势：系统学习
- 方法：参加培训/课程
- 时间：3-12个月准备

路径3：创业/自由职业
- 优势：完全掌控
- 方法：小步试错
- 时间：从副业开始
```

## 六、薪资谈判

### 谈判准备

#### 准备1：了解市场价值
```
调研方法：
1. 招聘网站（猎聘、BOSS直聘）
2. 行业报告（薪酬白皮书）
3. 人际网络（同行询问）

记录信息：
- 职位名称
- 公司规模
- 经验要求
- 薪资范围
```

#### 准备2：明确目标
```
薪资组成：
1. 基本工资
2. 绩效奖金
3. 股票期权
4. 福利待遇
   - 保险/公积金
   - 假期
   - 培训
   - 其他福利

目标设定：
- 期望薪资（合理范围）
- 接受底线（最低可接受）
- 谈判空间（弹性）
```

---

### 谈判技巧

#### 技巧1：让对方先报价
```
好处：
- 避免报价过低
- 了解公司预算
- 保留后续空间

策略：
- 询问："薪资范围是多少？"
- 等待对方回答
- 不急于表态
```

#### 技巧2：价值论证
```
使用数据：
- 过去业绩
- 带来的价值
- 市场价值

话术示例：
"根据我的[经验]、[技能]，以及对[业绩]的贡献，我认为期望薪资是合理的。"
```

#### 技巧3：非金钱谈判
```
如果薪资已定：
- 工作时间（远程/灵活）
- 职位名称（更高头衔）
- 学习机会（培训、认证）
- 工作地点（优选地点）
- 股票期权（长期收益）
```

## 七、常见问题

### Q1：如何选择职业？
**A：**
- 结合兴趣和优势
- 考虑市场需求
- 小步试错
- 长期规划

### Q2：没有经验怎么办？
**A：**
- 从实习/兼职开始
- 展示相关技能
- 完成个人项目
- 建立人脉

### Q3：职场遇到不公怎么办？
**A：**
- 保持专业记录
- 与上级沟通
- 了解劳动法规
- 必要时寻求法律帮助

### Q4：如何保持竞争力？
**A：**
- 持续学习
- 跟踪行业趋势
- 建立人脉
- 保持创新思维
