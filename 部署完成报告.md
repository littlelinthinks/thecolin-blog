## 🔧 故障排查

### 问题1: 网站无法访问
**可能原因**:
- Vercel 部署未完成
- DNS 配置未生效
- 缓存问题

**解决方案**:
1. 检查 Vercel Dashboard 部署状态
2. 等待 DNS 传播（最多 48 小时）
3. 清除浏览器缓存（Ctrl/Cmd + Shift + R）

### 问题2: 功能不工作
**可能原因**:
- common.js 或 common.css 未加载
- 浏览器缓存旧版本

**解决方案**:
1. 打开开发者工具 > Network
2. 检查 js/common.js 和 css/common.css 是否加载成功
3. 硬刷新（Ctrl/Cmd + Shift + R）
4. 清除浏览器缓存

### 问题3: 样式错乱
**可能原因**:
- common.css 路径错误
- CSS 冲突

**解决方案**:
1. 检查 HTML 中 `<link>` 标签路径
2. 查看控制台是否有 CSS 加载错误
3. 验证 CSS 变量是否正确定义
