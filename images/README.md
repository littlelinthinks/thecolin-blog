# 图片文件说明

将以下图片放入此文件夹：

## 必需图片

1. **wechat-qr.png** - 微信收款二维码
   - 尺寸：建议 400x400px
   - 格式：PNG
   
2. **alipay-qr.png** - 支付宝收款二维码
   - 尺寸：建议 400x400px
   - 格式：PNG

3. **og-image.jpg** - 社交分享预览图
   - 尺寸：1200x630px (推荐)
   - 格式：JPG/PNG
   - 用途：当别人在Twitter、Facebook等分享你的网站时显示

## 可选图片

4. **colin-photo.jpg** - 个人头像
   - 尺寸：建议 500x500px
   - 格式：JPG/PNG
   - 用途：About页面展示

5. **favicon.ico** - 网站图标
   - 尺寸：32x32px 或 16x16px
   - 格式：ICO
   - 用途：浏览器标签页图标

## 如何使用

图片准备好后，在 index.html 中更新相应路径即可。
