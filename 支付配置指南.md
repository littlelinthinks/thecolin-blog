## 方法1：支付宝收款码（10分钟）

### 步骤1：生成收款码
1. 打开支付宝APP
2. 点击"收钱"
3. 点击"二维码收款"
4. 保存二维码图片到相册
5. 点击"设置" → 可设置收款备注

### 步骤2：创建支付页面
1. 访问：https://render.alipay.com/p/f/alipay-new-qrcode/
2. 上传收款二维码图片
3. 设置收款金额（可选）
4. 生成收款链接
5. 复制支付链接

### 步骤3：测试支付
1. 用另一个支付宝账号扫二维码
2. 支付小额（如¥0.01）测试
3. 确认收款成功
4. 退款测试金额

### 自动计算公式
```excel
总收入 =SUM(金额列)
今日收入 =SUMIF(日期列, TODAY(), 金额列)
本周收入 =SUMIFS(金额列, 日期列, ">="&WEEKNUM(TODAY())-1)
本月收入 =SUMIFS(金额列, 日期列, ">"&EOMONTH(TODAY(),-1))
```
