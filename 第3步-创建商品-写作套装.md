### 步骤3.5：上传资源文件

#### 3.5.1 准备资源包

在电脑上准备ZIP压缩包，包含：
```
写作套装资源包.zip
├─ 01_写作模板大全/
│   ├─ 观点文模板.docx
│   ├─ 干货文模板.docx
│   ├─ 情感文模板.docx
│   └─ 故事文模板.docx
├─ 02_爆款标题库/
│   └─ 爆款标题100+.xlsx
├─ 03_Newsletter运营指南.pdf
├─ 04_Notion写作模板/
│   └─ 写作管理系统模板.zip
├─ 05_30天行动指南.pdf
├─ 06_使用说明.docx
└─ README.txt
```

**打包要求：**
- 使用ZIP格式
- 总大小 ≤ 500MB
- 文件命名清晰
- 包含README说明文件

#### 3.5.2 上传步骤

1. 找到「添加文件资源」区域
2. 点击「上传文件」
3. 选择准备好的ZIP文件
4. 等待上传完成（取决于文件大小和网络速度）
5. 上传成功后会显示文件名和大小

#### 3.5.3 设置文件权限

```
文件可见性：
☑️ 支付后可见（推荐）
□ 免费（不推荐）

下载次数限制：
□ 不限制
☑️ 限制每人下载3次（推荐）
```

**说明：**
- "支付后可见"确保只有付款用户才能下载
- "限制下载次数"可以防止资源被滥用
- 建议设置3次下载机会

## ⚠️ 常见问题

### Q1：资源文件大小超过500MB怎么办？
**A：**
- 压缩文件
- 分拆成多个文件
- 或使用网盘链接（在商品详情中提供）

### Q2：如何修改已发布的商品？
**A：**
1. 进入「商品管理」→「我的商品」
2. 点击商品名称
3. 点击「编辑」
4. 修改后保存

### Q3：自动发货邮件可以自定义吗？
**A：**
可以。在「发货设置」中编辑邮件模板。

### Q4：可以设置限时优惠吗？
**A：**
可以。在「营销工具」中创建限时优惠券。

### Q5：如何查看购买记录？
**A：**
进入「数据」→「订单管理」查看所有订单。

### Q6：商品发布后可以下架吗？
**A：**
可以。在商品管理中点击「下架」。

### Q7：如何设置推荐商品？
**A：**
在商品编辑页面，选择「推荐商品」标签。

### Q8：邮件模板支持HTML格式吗？
**A：**
支持。可以使用富文本编辑器添加格式。
