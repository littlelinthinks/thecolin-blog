# 目标套装 - 使用说明

## 📦 产品清单

本套装包含3个产品：

2. ✅ 年度规划PDF模板.md（22KB）

## 🚀 快速开始

### 方案1：使用Notion（推荐）

**适合人群：**
- 喜欢数字化管理
- 需要跨设备同步
- 想要自动化计算

**使用步骤：**
1. 打开"Notion目标系统-完整模板.md"
2. 复制模板内容
3. 在Notion中创建新页面
4. 粘贴模板
5. 根据需要自定义

---

### 方案2：使用PDF（打印）

**适合人群：**
- 喜欢手写记录
- 想要纸质版
- 习惯线下规划

**使用步骤：**
1. 打开"年度规划PDF模板.md"
2. 导出为PDF（或直接打印）
3. 打印10页（A4纸）
4. 手写填写
5. 或导入到GoodNotes/Notability

---

### 方案3：使用Excel

**适合人群：**
- 喜欢数据可视化
- 需要自动计算
- 想要统计分析

**使用步骤：**
1. 打开"Excel追踪表-模板说明.md"
2. 新建Excel文件
3. 按照说明创建6个工作表
4. 复制公式
5. 设置数据验证
6. 开始使用

---

## 💡 推荐使用组合

**方案A：Notion + PDF**
- 日常使用Notion（随时随地更新）
- 重要节点打印PDF（深度规划）

**方案B：Excel + Notion**
- Excel做统计分析（数据看板）
- Notion做详细记录（内容管理）

**方案C：三种工具都用**
- PDF：年度规划（打印出来放桌上）
- Notion：日常管理（手机电脑同步）
- Excel：统计分析（月末复盘）
