# 第2步：配置支付方式

## 🎯 目标
开通微信支付和支付宝支付，设置结算规则，确保能正常收款。

**预计时间：** 20分钟
**难度：** ⭐⭐☆☆☆

---

## 📋 准备工作

### 需要准备的材料
- ✅ 本人银行卡
- ✅ 银行卡开户行信息
- ✅ 银行卡预留手机号
- ✅ 身份信息（已实名）

### 支付开通顺序
建议按以下顺序开通：
1. 支付宝支付（审核快，1小时内）
2. 微信支付（审核稍慢，1-3个工作日）

---

## 💳 步骤2.1：开通支付宝支付

### 2.1.1 进入支付设置页面

1. 登录小鹅通账号
2. 点击顶部菜单「店铺」
3. 在左侧菜单找到「支付设置」
4. 点击进入

### 2.1.2 找到支付宝支付

在支付设置页面，你会看到：
```
支付方式
├─ 微信支付 [未开通]
└─ 支付宝支付 [未开通]  ← 点击这里
```

点击「支付宝支付」后面的「立即开通」按钮

### 2.1.3 填写支付宝商户信息

#### 商户类型选择
选择：个人

#### 个人信息填写

```
支付宝账号：_____________________
  • 填写你的支付宝账号
  • 可以是手机号或邮箱
  • 确保账号正常且已实名

真实姓名：_____________________
  • 必须与支付宝账号实名一致
  • 与小鹅通实名一致
  • 系统可能自动填充

身份证号：_____________________
  • 18位身份证号码
  • 系统自动填充

手机号：_______________________
  • 支付宝绑定的手机号
  • 用于接收验证码
  • 与注册账号一致
```

#### 经营信息填写

```
店铺名称：_____________________
  • 建议使用你的品牌名
  • 示例：小Lin成长笔记
  • 或：个人成长资源库

经营类目：_____________________
  • 选择：教育培训
  • 或：知识服务
  • 或：咨询服务

经营范围：_____________________
  • 描述你的业务范围
  • 示例：个人成长类数字产品销售
  • 或：写作、学习、目标管理等课程和资源
```

#### 结算信息填写

```
结算账户：_____________________
  • 选择：支付宝账户
  • 支付宝账号同上
  • 用于接收货款

开户银行：_____________________
  • 绑定的银行卡开户行
  • 示例：招商银行北京分行
  • 或：中国工商银行XX支行

银行卡号：_____________________
  • 支付宝绑定的银行卡
  • 用于账户验证
  • 可以不是结算账户

开户人姓名：__________________
  • 银卡卡主姓名
  • 必须与支付宝实名一致
  • 系统可能自动填充
```

### 2.1.4 上传必要资料

根据要求上传：
- ✅ 身份证正反面（如需要）
- ✅ 营业执照（个人可跳过）
- ✅ 其他证明材料（按提示）

### 2.1.5 提交审核

1. 检查所有信息是否正确
2. 确认无误后点击「提交审核」
3. 看到"审核已提交，预计1小时内完成"
4. 等待审核结果

### 2.1.6 查看审核状态

审核状态查看位置：
- 短信通知
- 支付设置页面

状态显示：
- ⏳ 审核中（正常）
- ✅ 已通过（成功）
- ❌ 审核失败（需要重新提交）

## ⚙️ 步骤2.3：设置结算规则

### 2.3.1 进入结算设置

1. 点击「店铺」→「结算设置」
2. 查看当前结算配置

### 2.3.2 确认结算周期

建议设置：
```
结算周期：T+1（第二天到账）

说明：
• 用户今天支付，明天就能收到货款
• 流动性最好
• 适合日常经营
```

### 2.3.3 设置起提金额

建议设置：
```
起提金额：¥100

说明：
• 账户余额达到¥100才能提现
• 避免频繁小额提现产生手续费
• 如果需要立即提现，可以设置¥1

考虑：
• 小额交易多 → 设置¥50
• 中等交易 → 设置¥100（推荐）
• 大额交易 → 设置¥200
```

### 2.3.4 查看手续费

确认手续费设置：
```
支付手续费：
✅ 微信支付：0.6%
✅ 支付宝支付：0.6%
✅ 银行卡支付：0.6%

提现手续费：
✅ T+1提现：免费
✅ T+0提现：0.1%（次日到账 vs 当日到账）
```

### 2.3.5 保存设置

1. 确认所有设置
2. 点击「保存」或「确认」
3. 看到提示"设置已保存"

## ⚠️ 常见问题

### Q1：支付宝审核多久能通过？
**A：**
- 通常：30分钟内
- 最快：5分钟
- 最长：1小时

### Q2：微信支付审核比支付宝慢，正常吗？
**A：**
正常。微信支付审核需要人工审核，通常1-3个工作日。

### Q3：结算周期T+1是什么意思？
**A：**
T+1表示交易日的第二天到账。
- 用户今天（T日）支付
- 资款明天（T+1日）到账
- T+0可以当日到账，但有0.1%手续费

### Q4：手续费可以降低吗？
**A：**
0.6%是微信支付和支付宝的官方标准费率，个人商户无法降低。企业商户达到一定流水后可以申请更低费率。

### Q5：审核失败怎么办？
**A：**
1. 查看失败原因
2. 修正错误信息
3. 重新上传资料
4. 重新提交审核
5. 可以联系小鹅通客服协助

### Q6：可以用别人的银行卡收款吗？
**A：**
不可以。结算账户的银行卡开户人必须与实名认证一致，否则会被驳回。

### Q7：起提金额设置后可以改吗？
**A：**
可以。随时进入结算设置修改，立即生效。

### Q8：手续费是每笔交易都收吗？
**A：**
是的。每笔成功交易都会收取0.6%手续费。

### Q9：提现有手续费吗？
**A：**
T+1提现免费，T+0提现（当日到账）收取0.1%手续费。建议选择T+1。

### Q10：可以同时开通多个支付方式吗？
**A：**
可以，而且建议都开通。微信和支付宝可以同时使用，方便用户选择。

## 🔄 审核失败常见原因

### 原因1：实名信息不一致
- 解决：确保小鹅通、支付宝、微信实名信息一致

### 原因2：结算账户信息错误
- 解决：检查银行卡号、开户行是否正确

### 原因3：资料照片不清晰
- 解决：重新上传清晰的照片

### 原因4：经营范围描述不当
- 解决：使用标准描述，如"教育培训"、"知识服务"

### 原因5：经营类目选择错误
- 解决：选择"教育培训"或"咨询服务"
