# 思维套装 - 21天思维训练计划

## 🎯 你的目标

完成本计划后，你将能够：
- ✅ **第7天**：掌握第一性原理，学会深度分析问题
- ✅ **第14天**：熟练使用逆向思维，发现隐藏机会
- ✅ **第21天**：建立系统思考框架，提升决策质量

---

## 📋 第一周：第一性原理（Day 1-7）

### Day 1：认识第一性原理（时间：60分钟）

**📚 学习任务：**
- [ ] 阅读《第一性原理入门》PDF（30分钟）
- [ ] 理解核心：把复杂问题拆解到最基础（15分钟）
- [ ] 记录3个现实生活案例（15分钟）

**💡 今日重点**：所有复杂问题都有最简单的起点

**📝 练习**：
用第一性原理分析"如何提升阅读量"：
- 问：阅读量的本质是什么？
- 答：更多人看到 + 更多人愿意读
- 拆解：更多人看到 = 分发渠道 + 推荐机制
- 拆解：更愿意读 = 内容价值 + 情绪共鸣

---

### Day 2：案例拆解（时间：90分钟）

**📚 任务清单：**
- [ ] 选择一个你关心的复杂问题（10分钟）
  - 例如：如何提升工作效率？
  - 如何赚到更多钱？
- [ ] 连续问"为什么"5次，找到本质（40分钟）
- [ ] 从本质出发，重新思考解决方案（40分钟）

**💡 今日重点**：不要直接跳到解决方案

**📝 输出**：完成1个问题的第一性原理分析

## 📋 第二周：逆向思维（Day 8-14）

### Day 8：认识逆向思维（时间：60分钟）

**📚 学习任务：**
- [ ] 阅读《逆向思维指南》PDF（30分钟）
- [ ] 理解核心：从相反方向思考（20分钟）
- [ ] 记录3个经典案例（10分钟）

**💡 今日重点**：反向思考能看到正向看不到的

**📝 练习**：
正向：如何成功运营一个账号？
逆向：如何让一个账号死掉？
答案：
- 不更新内容
- 内容无聊
- 互动差
- ...
→ 反过来就是成功要素

**📝 输出**：一份"成功避坑指南"

## 📋 第三周：系统思考（Day 15-21）

### Day 15：认识系统思考（时间：60分钟）

**📚 学习任务：**
- [ ] 阅读《系统思考框架》PDF（30分钟）
- [ ] 理解核心：事物是相互关联的（20分钟）
- [ ] 画一个简单的系统图（10分钟）

**💡 今日重点**：没有孤立的因果

**📝 练习**：
问题：为什么总睡不够？
系统图：
- 工作忙 → 晚睡 → 累 → 效率低 → 工作忙（循环）
→ 打破循环：提升效率，减少工作时间

---

## 🆘 常见问题

### Q1: 不知道该用哪个方法怎么办？
**A**: 
- 第一性原理：适合分析新问题，找本质
- 逆向思维：适合避免失败，发现盲点
- 系统思考：适合复杂、循环的问题
- 不确定就都试试

### Q2: 思考没深度怎么办？
**A**:
- 多问几次"为什么"
- 强制自己想3个不同角度
- 在社群讨论碰撞观点

### Q3: 想不出怎么办？
**A**:
- 从小问题开始
- 查看案例库找启发
- 问朋友："你会怎么想？"
