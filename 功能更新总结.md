# 网站功能更新总结

## ✅ 已完成的功能

### 🔴 高优先级（已实现）

#### 1. ✅ RSS Feed 自动生成
- **文件**: `generate_rss.py`
- **功能**: 从 `articles.json` 自动生成 RSS Feed
- **特点**:
  - 自动解析中文和英文日期格式
  - 支持文章标题、链接、描述、分类

- **使用**: `python3 generate_rss.py`

#### 2. ✅ Sitemap 自动生成

## 🚀 使用指南

### 添加新文章
1. 在 `articles.json` 中添加文章信息
2. 将 Markdown 文件放入 `articles/` 目录
3. 运行 `python3 generate_rss.py` 更新 RSS
4. 运行 `python3 generate_sitemap.py` 更新 Sitemap
5. 运行 `./update-all.sh` 一键部署

### 更新网站
```bash
# 方式1: 完整部署
./update-all.sh

# 方式2: 分步操作
python3 generate_rss.py
python3 generate_sitemap.py
git add -A
git commit -m "Update: 描述"
git push
```
