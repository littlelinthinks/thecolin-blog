## 📱 注册配置

### 步骤1：注册小鹅通（5分钟）

1. 访问：https://xiaoe-tech.com/
2. 点击「免费注册」
3. 选择「个人创作者」
4. 输入手机号和验证码
5. 设置密码并登录

### 步骤2：实名认证（10分钟）

**路径：** 设置 → 实名认证

需要准备：
- ✅ 身份证正反面照片
- ✅ 进行人脸识别
- ✅ 填写真实信息

### 步骤3：配置支付（10分钟）

#### 开通微信支付
**路径：** 店铺 → 支付设置 → 微信支付

1. 点击「立即开通」
2. 填写商户信息
3. 上传结算资料
4. 审核时间：1-3个工作日

#### 开通支付宝支付
**路径：** 店铺 → 支付设置 → 支付宝支付

1. 点击「立即开通」
2. 填写支付宝账号
3. 绑定银行卡
4. 审核时间：1小时内

### 步骤4：设置结算规则

**路径：** 店铺 → 结算设置

- 结算周期：T+1（推荐）
- 起提金额：¥100
- 手续费：0.6%（微信/支付宝）

### 方法2：添加支付说明弹窗

在页面底部添加支付说明：

```html
<!-- 支付说明弹窗 -->
<div id="payment-modal" style="display: none; position: fixed; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0,0,0,0.5); z-index: 9999;">
    <div style="position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); background: white; padding: 40px; border-radius: 20px; max-width: 500px; width: 90%;">
        <h3 style="margin-bottom: 20px;">💳 支付说明</h3>
        <p style="color: #666; margin-bottom: 15px;">
            我们使用小鹅通安全支付平台，支持：
        </p>
        <ul style="margin-bottom: 20px; color: #666;">
            <li>✅ 微信支付</li>
            <li>✅ 支付宝支付</li>
            <li>✅ 银行卡支付</li>
        </ul>
        <p style="color: #666; margin-bottom: 20px;">
            购买后30分钟内自动发送资源到您的邮箱
        </p>
        <div style="display: flex; gap: 10px;">
            <button onclick="closePaymentModal()" style="flex: 1; padding: 12px; background: #f0f0f0; border: none; border-radius: 8px; cursor: pointer;">取消</button>
            <button onclick="confirmPayment()" style="flex: 1; padding: 12px; background: #667eea; color: white; border: none; border-radius: 8px; cursor: pointer;">去支付</button>
        </div>
    </div>
</div>

<script>
function openPaymentModal(link) {
    document.getElementById('payment-modal').style.display = 'block';
    window.currentPaymentLink = link;
}

function closePaymentModal() {
    document.getElementById('payment-modal').style.display = 'none';
}

function confirmPayment() {
    if (window.currentPaymentLink) {
        window.open(window.currentPaymentLink, '_blank');
    }
    closePaymentModal();
}
</script>
```
