## 📋 第一周：费曼学习法（Day 1-7）

### Day 1：认识费曼学习法（时间：60分钟）

**📚 学习任务：**
- [ ] 阅读《费曼学习法指南》PDF（30分钟）
- [ ] 理解核心：用简单语言教会别人 = 真懂（20分钟）
- [ ] 记录费曼学习4步骤（10分钟）

**💡 今日重点**：如果不能用简单语言解释，就是没真懂

**📝 练习**：
选择一个你"以为自己懂"的概念（如：什么是区块链？）
- 尝试用初中生能懂的语言解释
- 发现解释不清的地方，就是知识盲区

## 🆘 常见问题

### Q1: 没时间每天学习怎么办？
**A**: 
- 周一到周五每天20分钟，周末2小时
- 利用碎片时间：通勤、午休
- 目标是"持续"而不是"完美"

### Q2: 记不住怎么办？
**A**:
- 检查：有没有用间隔重复？
- 检查：有没有用费曼法真正理解？
- 检查：有没有输出应用？

### Q3: 学不会怎么办？
**A**:
- 换个资源（视频/书/文章）
- 找人问（社群/论坛）
- 降难度：先学基础，再进阶

## 🎯 学习方法速查卡

### 🎓 费曼学习法
**何时用**：深度理解新概念
**核心口诀**：我能用简单语言教会别人吗？
**关键动作**：讲解 → 找盲区 → 补救 → 再讲解

### 🔄 间隔重复
**何时用**：记忆知识点、外语单词
**核心口诀**：在遗忘前复习
**关键动作**：制作卡片 → 定时复习 → 记录成功率

### 📖 深度阅读
**何时用**：学习重要知识/专业书籍
**核心口诀**：先浏览，再精读，要输出
**关键动作**：浏览判断 → 精读理解 → 输出应用
